Accounting Assistant

Analyze images of receipts and invoices to extract line items, vendors, and total amounts, securely saving them to a local in-browser database.

Learn How It Works

Local Processing for Financial Documents

The Accounting Assistant acts as a local processor. By supplying a photo of a receipt or invoice, or directly describing an expense, the AI uses structured data extraction to capture core details like the vendor, total amount, date, and line items. The assistant then stores this information securely in your device's local database using an SQLite abstraction over IndexedDB via WebAssembly (sql.js).

Core Capabilities

Key Features

Automated Data Extraction

Transforms pictures of physical receipts or invoices into clean, structured JSON format effortlessly.

Local Embedded Database

Uses a WebAssembly-compiled SQLite database backed by IndexedDB. Your data never improperly leaks to the cloud; it all lives on the edge device.

Web Dashboard Integration

Renders a rich HTML view to beautifully display your saved expenses and receipts directly on your mobile device.

How It Works

1

Input

Send an image of a receipt, an image of an invoice, or text describing an expense to the AI.

2

Analysis

The AI model processes the input and calls an underlying execution tool (run_js) with extracted parameters:

vendor total date items list
3

Execution

A script intercepts this structured JSON. It initializes an SQLite table mapped to IndexedDB and executes a SQL INSERT statement with your parsed receipt data.

4

Display

On successful entry, the tool returns the path to webview.html, providing an interface to visualize or manage saved expenses.

extracted_data.json
{

Architecture & Layout

SKILL.md Defines the model persona, execution parameters, and strict structures for the JSON payload required.
scripts/index.html Execution environment. Loads sql-wasm.js and provides loadFromIndexedDB / saveToIndexedDB abstractions.
scripts/sql-wasm.js & .wasm The WebAssembly SQLite port for local database operations.
assets/webview.html The interactive front-end visualizer for your saved expenses.

Requirements

  • This skill requires the Google Edge AI application host.
  • Browser support for IndexedDB.
  • Browser support for WebAssembly to run the local SQLite database.
